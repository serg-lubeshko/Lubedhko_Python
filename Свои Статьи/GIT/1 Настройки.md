# Базовая настройка

|                                        |                                                         |      |
| -------------------------------------- | ------------------------------------------------------- | ---- |
| git --version                          | Посмотреть версию                                       |      |
| git config --global user.name "FIO"    | Установить имя (без global будут локальные настройки)   |      |
| git config --global user.email "EMAIL" | Установить почту (без global будут локальные настройки) |      |
| git config --list                      | Посмотреть список настроек                              |      |
| git --help                             | Система подсказок                                       |      |
| git log --help                         | Конкретная справка по функции в html                    |      |
|                                        |                                                         |      |
|                                        |                                                         |      |
|                                        |                                                         |      |
