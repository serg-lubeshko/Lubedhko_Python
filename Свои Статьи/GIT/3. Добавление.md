# Добавление файла

|                                 |                                                              |
| ------------------------------- | ------------------------------------------------------------ |
| git add -f <.idea/project.>     | Добавление принудительное если папка в git ignore            |
| git add -p <index.html>         | Если хотим добавить выборочные строки в файле он спросит. Щелкаем y or n.  Но ВНИМАНИЕ игнорит не отслеживаемые файлы |
| git commit -am "message"        | Команда объединяет add . и commit. Но ВНИМАНИЕ игнорит не отслеживаемые файлы |
| git commit -m "text" .gitignore | Если срочно надо внести изменения в файл и его отправить в репу. НЕ ТРОГАЕМ другие файлы, которые ждут добавления в гит. Проходит по по рабочей>локальной>удаленный репо. |
|                                 |                                                              |
|                                 |                                                              |
